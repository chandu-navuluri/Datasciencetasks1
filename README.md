# DS_SCT_01 - Population Distribution Visualization

## 📌 Task Description
Create a bar chart or histogram to visualize the distribution of a categorical or continuous variable, such as the distribution of ages or genders in a population.

This project is part of the **SkillCraft Technology Data Science Internship**.

---

## 🎯 Objective
Analyze population data and create meaningful visualizations to understand the distribution of different demographic groups.

---

## 📊 Dataset
Dataset Used:
- World Bank Population Dataset / Sample Population Dataset

Features:
- Country Name
- Population
- Year
- Other demographic attributes

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook / VS Code

---

## 📈 Visualization
The project generates:
- Bar Chart for population distribution
- Histogram for age/group distribution

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/chandu-navuluri/DS_SCT_01.git
```

2. Navigate to the project folder

```bash
cd DS_SCT_01
```

3. Install required libraries

```bash
pip install pandas matplotlib seaborn
```

4. Run the script

```bash
python task1.py
```
